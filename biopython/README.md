# Biopython

Scripts using Biopython.

## Setup

Use `create_venv.sh` to create a virtual environment.

```console
./create_venv.sh
```

Then activate it.

```console
source venv/bin/activate

which python3
# ~/github/learning_python/biopython/venv/bin/python3
```

Deactivate once you have finished.

```console
deactivate
```
